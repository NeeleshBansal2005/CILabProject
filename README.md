# CI/CD Pipeline Automation Lab

**Author:** Neelesh Bansal
**Date:** February 1, 2026

## Project Overview
This repository demonstrates a fully automated CI/CD pipeline using **Jenkins**, **Maven**, and **Docker**. The pipeline automates the software delivery process, ensuring that every commit is built, tested, and containerized for deployment.

## Tech Stack
* **Source Code Management:** Git & GitHub
* **CI Server:** Jenkins
* **Build Tool:** Apache Maven
* **Containerization:** Docker
* **Language:** Java
